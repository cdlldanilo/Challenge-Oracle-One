# Challenge ONE | Java | Back-end | Hotel Alura

##  Sobre o Projeto:

## Pré - Requisitos:

- jdk11
- jre11
- eclipse
- windows
- mySql8.0 
- mySql workbench 8.0 C.E

## Projeto: realizado como desafio durante o programa ONE-Alura com o nome de Hotel-Alura. Durante esse desafio foi disponibilizado uma interface com a tecnologia SWING do java, e o participante tinha que realizar a parte back-end. Foi usado o padrões MVC, PreparedStatement e DAO.
